## ✨ 주요 기능

### 🔮 예측 엔진
- **11가지 조건** 기반 `rawH` 생성 → **S1~S9 고급 파이프라인** 처리
- **xG 근사** — 팀 공격력 × 상대 수비취약도 / 리그평균
- **포아송 분포** — xG를 λ로 사용, 0~6골 시뮬레이션 → 홈/무/원정 확률
- **Elo 레이팅** — 경기 결과 입력 시 자동 갱신
- **H2H 상대전적** — GW32~36 전 50경기 100% 커버
- **포메이션 상성** — MATCHUP_MATRIX (6×6)
- **심판 데이터** — EPL 주요 심판 10명 성향 DB 내장
- **일정 피로도** — API 경기일정 기반 자동 계산

### 🤖 AI 조합픽
- **GW별 독립 생성** (절대 라운드 혼합 없음)
- **4가지 필터**: `conf ≥ 60%`, `|H-A| ≥ 10%`, `D < 32%`, `|rawH-50| ≥ 6`
- **재귀(DFS) 조합** 생성 → Top-K 제한
- **3가지 타입**: 🛡️ 안전 / ⚡ 밸류 / 🎯 이변
- **Value 계산**: `(AI확률 × 배당) - 1`
- **기대수익(EV)**: `성공률 × 배당`

### 📊 분석 기능
- **예측 근거 카드** — 승점차·폼·H2H·xG·Elo 수치별 기여도
- **이변 감지** — rawH 높은데 원정 xG 우위 시 ⚠️ 경고
- **백테스트** — GW1~31 310경기 시뮬레이션, 신뢰도 구간별 적중률
- **캘리브레이션** — 백테스트 오차 자동 보정

### 📱 UX
- **경기 결과 직접 입력** → 예측 기록 자동 판정 + Elo·폼 갱신
- **분석 공유** — 클립보드 복사 (SNS 공유용)
- **모바일 스와이프** 탭 전환
- **GW 자동 감지** — 현재 날짜 기준 라운드 자동 선택
- **배당 시장 연동** — The Odds API (모델 75% + 시장 25%)

## 🧮 예측 공식 요약

```
rawH = 48.0 (기준)
     + 승점차 ÷ 5
     + 득실차 × 0.055
     + 폼 × 0.85
     + 홈어드밴티지 +8
     + H2H 보정 × 0.12
     + GW 시즌 패턴
     + 포메이션 상성 × 0.55
     + 스쿼드 강도 × 0.14
     + Elo 차이 × 0.015
     + 홈/원정 특화 승률 × 12
     + 피로도 ±2%
     + xG 보정 × 6.5
     + 심판 홈편향 ±2%
     → 배당편향 보정
     → S1 로지스틱(k=0.09)
     → S3 무승부(xG균형+전술+총xG+승점차)
     → S4 시너지 보정
     → S5 배당시장 보정
     → S6 재정규화
     → S7 EPL 앵커링
     → 포아송 혼합(모델70%+포아송30%)
     → 백테스트 캘리브레이션
```

## 📈 백테스트 성능

| 신뢰도 구간 | 경기 수 | 적중률 |
|---|---|---|
| 65% 이상 (확실) | 8경기 | **88%** |
| 55~64% (우세) | 75경기 | **61%** |
| 45~54% (근소) | 96경기 | **55%** |
| 35~44% (박빙) | 92경기 | 48% |
| **전체** | **310경기** | **52.3%** |

> ⚠️ 백테스트는 실제 EPL 경기 결과가 아닌, 각 팀 W/D/L 예산 기반 시뮬레이션 결과입니다.

## 🚀 사용 방법

### 설치 없이 바로 사용
```
index.html 파일을 브라우저에서 열기
```
끝입니다. 서버, 빌드, 설치 전혀 없어요.

### API 연동 (선택사항)

실시간 순위·폼·일정 데이터를 위해 API 키를 설정하면 정확도가 올라가요.

**football-data.org** (무료, 10회/분)
1. [football-data.org/client/register](https://www.football-data.org/client/register) 가입
2. 사이트 접속 → 헤더 **⚙ API** 버튼 → 키 입력
3. **새로고침** 버튼으로 순위·폼·일정 자동 갱신

**The Odds API** (선택, 무료 500회/월)
1. [the-odds-api.com](https://the-odds-api.com) 가입
2. 헤더 **💰 배당** 버튼 → 키 입력
3. 배당 시장 데이터가 예측에 자동 반영됨 (정확도 +3~5% 예상)
